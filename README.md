Installation process
   * Prerequisites: 
    - Node v6.3.1 or later
    - NPM v3.10.3 or later

   * Install: npm install -g create-react-app
   * Run: npm install

Running the project: 
  npm run start

Build with prod settings: 
  npm run build

Run tests: 
  npm run test
